Body text uses a 16px base size with 1.462 line-height. Paragraphs use 1.57 line-height for comfortable reading. Links appear in the foreground color and change to accent blue on hover.

Small text at 14px is used for captions, labels, and secondary information.

Spacing

Content uses an 800px max-width for readability. Wide content extends to 1360px. Full-width elements span the entire viewport.

Section Padding

  • Large sections: 60-100px vertical
  • Standard sections: 40px vertical
  • Horizontal padding: 20px (mobile/tablet)

Element Gaps

  • Block gap: 20px default
  • Paragraph margin: 30px bottom
  • Heading margin: 16px bottom
